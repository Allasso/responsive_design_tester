Setting Viewport Dimensions

There are 4 dimensions which affect the viewport rendering:
  • css width:  The width in css pixels of the viewport
  • css height:  The height in css pixels of the viewport
  • phys size:  The physical size of the display, measured diagonally in inches
  • phys width:  The physical width of the display, measured across in inches
phys width is calculated from phys size, using the aspect ratio obtained from css width and css height. If either css width, css height, or phys size is changed, phys width is recalculated.
Note that manually changing phys width value will conversely recalculate phys size. Note also that changing either of these values will never affect css width nor css height.
Also note that css width and css height are logical pixels, and may or may not be the same as physical pixels of a device display.

Calibration

In order to view the viewport in actual physical size, the simulator must be calibrated to your browser/display.
Enter a value or use the thumbwheel to the left of the ruler to adjust the ruler until it is exactly 3 physical inches (tip: a common "post-it" sticky note is exactly 3" - you can stick it just above the ruler.)
You can record the pixel value from the number input for future uses.
This method is preferred to asking user's screen size, as that method is prone to error.

Font Size

If text is selected within the displayed page, this will attempt to force font size of the selected text to the input value. The value is in pixels.
If "parent" is checked, the font size will be applied to the closest ancestor container of the selected text. e.g., if a portion of text in a paragraph is selected, font size may be applied to the entire paragraph. If the selection spans two paragraphs, font size may be applied to all paragraphs.
Refresh page by clicking "go" to clear applied font sizes.

Image Size

If image is selected within the displayed page, this will attempt to force the image size to the input value.
Enter two values, width followed by height separated by a delimiter, which can be a space or anything except a number, eg:
120x200, 120 200, 120,200.
Decimal px values not allowed, and decimal point will act as a delimeter.
Refresh page by clicking "go" to clear applied sizes.
